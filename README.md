# vegasblackjack

a tool designed to support and optimize training for 'Basic Strategy' approach to blackjack along with a card counting system to optimize blackjack return to player (RTP) potential in 3:2 blackjack. 

- decks customizable 1 to 8 decks (& frequency of deck reshuffle customizable to every 1 to 8 decks).
- basic strategy toggle to show or hide correct action for each hand.
- "re-bet mode" to automatically repeat last bet.
- start with $1,000 bankroll.
- bets denominated in $5 / $10 / $25 / $50 / $100.
- stats counted: Count / True Count / # of Hands / Win/Loss/Push / Net $ Gain.
